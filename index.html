Web API

Basado en los principios de REST, nuestros endpoints de la API web de Boom Percussion; regresan archivos JSON con informacion de las credenciales del usuario, estadisticas globales del videojuego, estadisticas personales del jugador y datos relacionados con el videojuego y el jugador.

Requests

Debido a que nuestra API se basa en principios REST. Podemos acceder a los recursos de datos a traves de solicitudes HTTPS al endpoint. Algunos de estos verbos son:

Metodo Accion
GET Recupera recursos
POST Crea recursos
PUT Cambia y/o reemplaza recursos
DELETE Elimina recursos
